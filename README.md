# FC Barcelona — Data Governance Scorecard
An interactive Power BI dashboard modeling what data governance maturity looks like for an organization the size of a major football club — tracking data quality, ownership, privacy compliance, and cross-department adoption in a single scorecard.

## Overview

Most data governance conversations stay abstract — "we need better data quality," "ownership needs to be clearer." This project turns those ideas into a measurable, visual scorecard: five governance pillars, each with concrete KPIs, built entirely in Power BI.

## Governance Pillars Tracked

- Data Quality & Reliability: Quality check pass rate, metadata completeness
- Stewardship & Ownership: Dataset coverage, owner vs. steward distribution
- Privacy & Compliance: Privacy incident rates by severity, datasets with no incidents
- Culture & Adoption: Governance participation by business unit, submission activity
- Analytics Readines: sShare of initiatives built on governed data

## Key Results


- 71.7% metadata completeness
- 71.7% of datasets with an assigned owner or steward
- 73.9% of initiatives using governed data
- 68.3% participation in governance activities across business units


## Tools & Techniques


- Power BI Desktop — data modeling, DAX measures, report design
- KPI card design, donut/bar visualizations for distribution and severity breakdowns
- Simulated multi-department dataset (HR, Finance, IT, Marketing, Analytics, Legal, Retail, Ticketing) to reflect realistic organizational scale
